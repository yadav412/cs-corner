Javascript Lesson 1

Part of the basics of web development alongside HTML and CSS. Can be run in the browser or server side using Node.js.

JavaScript is primarily a high-level scripting language.

Linking JavaScript

Inline in HTML:


<script>
    console.log("Hello there");
</script>

External JS file:


<script src="jsfile.js"></script>

Hello World

Output to console using console.log():


console.log("Hello there");

Variables


let age = 18;      // mutable
const name = "Yadav"; // immutable

Functions


function greet(name) {
    return `Hello, ${name}!`;
}
console.log(greet("Yadav"));

Loops

While Loop


let i = 0;
while(i < 5){
    console.log(i);
    i++;
}

For Loop


for(let i=0; i<5; i++){
    console.log(i);
}

For-of Loop


let fruits = ["apple", "banana"];
for(let fruit of fruits){
    console.log(fruit);
}

Events

Respond to user actions:


document.getElementById("myBtn").addEventListener("click", () => {
    alert("Button clicked!");
});
